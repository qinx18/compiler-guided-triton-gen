
\section{Evaluation Setup}

\cparagraph{Platforms.}
Experiments run on an NVIDIA RTX~3090 GPU (24\,GB, 936\,GB/s) with a dual-socket Xeon Gold 5218R CPU (80 cores). We use PyTorch~2.5.1, Triton~3.1.0, CUDA~12.4, and Claude Sonnet~4~\cite{anthropic2025claude}.

\cparagraph{Benchmarks.}
We evaluate on:
(1)~\emph{PolyBench/C}~\cite{polybench}: 30 kernels at default sizes ($N$=60--120) and 8$\times$ scale;
(2)~\emph{TSVC}~\cite{callahan1988tsvc}: 151 loop kernels at 32\,MB and 3200\,MB per array;
(3)~8 application kernels from Rodinia~\cite{che2009rodinia} and ECP proxy apps~\cite{heroux2009mantevo}.

\cparagraph{Baselines.}
We compare against OMP~GPU (OpenMP offloading via \texttt{nvc} on the same GPU), OMP~CPU (80-thread OpenMP), and single-threaded C (\texttt{-O3}).
For the ablation, we compare \emph{with analysis} (WA) vs.\ \emph{no analysis} (NA, LLM receives only C source).
